# .devcontainer

## Purpose

Devcontainer definitions for local contributor environments.

## Current Status

Active and backend-oriented. Configuration is being kept in sync with current Nexus repo direction.

## What Belongs Here

- `devcontainer.json` and compose overrides
- post-create bootstrap scripts
- contributor environment setup notes

## What Does Not Belong Here

- assumptions that removed frontend/demo paths still exist
- heavyweight setup steps that mutate global host settings unnecessarily

## Notes

Keep this folder aligned with active backend decisions and future frontend ADR direction.
