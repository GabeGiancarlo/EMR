# .cursor

## Purpose

Cursor AI guidance rules for repository-aware development behavior.

## Current Status

Active. Rules shape AI-assisted coding and documentation quality.

## What Belongs Here

- coding and architecture guidance rules
- repository-specific AI constraints
- maintainability and style guardrails

## What Does Not Belong Here

- unrelated project artifacts
- generated build output
- ad hoc notes without clear guidance value

## Notes

Rules should help prevent sloppy Rust, fake production claims, orphan files, demo-only implementation patterns, and unreviewed architecture drift.
