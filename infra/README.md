# infra

## Purpose

Infrastructure, database, container, and deployment reference assets for Nexus.

## Current Status

Reference/evolving. Useful for local and planning use, not fully production-ready.

## What Belongs Here

- container and compose definitions
- database initialization and migration assets
- monitoring/deployment configuration scaffolding
- environment templates and infrastructure notes

## What Does Not Belong Here

- product feature logic
- stale service definitions that do not match active workspace structure

## Notes

Infra should continue migrating toward Postgres + disciplined migrations + deployment configs aligned to active backend/frontend decisions.
